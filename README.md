# 📈 StockPicker Dashboard

Screener técnico + fundamentalista de ações americanas com sinais de compra/venda.

## Features
- ✅ Score técnico de confluência (RSI, MACD, ADX, Fibonacci, price action)
- ✅ Análise fundamentalista (P/L, P/VP, ROE, dividend yield, endividamento)
- ✅ Integração com TradingView-TA (26+ indicadores)
- ✅ 4 timeframes: Semanal, Diário, 1h, 30min
- ✅ Detecção de data do gatilho + variação % desde o sinal
- ✅ Backtest histórico

## Rodar localmente
​```bash
pip install -r requirements.txt
streamlit run app.py
​```
